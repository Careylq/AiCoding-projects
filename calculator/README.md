# 简易计算器

一个 iOS 风格的网页计算器，支持括号优先级运算与历史记录。

## 功能

- **四则运算**：加减乘除，自动按优先级计算（先乘除，后加减）
- **括号支持**：任意嵌套，自动补全缺失的右括号
- **高级运算**：百分比（`%`）、正负号切换（`±`）、小数点
- **隐式乘法**：`3(2+1)` 自动识别为 `3×(2+1)`
- **历史记录**：保留最近 20 条计算，可滚动查看，点击直接复用结果
- **键盘输入**：完整键盘支持，无需鼠标
- **响应式布局**：手机到桌面自适应，一屏内展示全部按钮

## 快速开始

直接双击打开 `index.html`，或用浏览器打开：

```sh
open calculator/index.html
```

## 操作方式

### 鼠标 / 触屏

点击按钮即可输入，功能和外观同 iOS 计算器一致。

### 键盘

| 按键 | 功能 |
|------|------|
| `0`–`9` | 输入数字 |
| `.` | 小数点 |
| `+` `-` `*` `/` | 四则运算符 |
| `(` `)` | 括号 |
| `%` | 百分比 |
| `Enter` / `=` | 求值 |
| `Backspace` | 退格删除 |
| `Esc` / `C` | 清空 (AC) |

## 边界处理

| 场景 | 行为 |
|------|------|
| 除以零 | 显示 Error，按任意键自动清除 |
| 连续运算符 | 覆盖上一个运算符 |
| 浮点精度 | 自动四舍五入到合理位数 |
| 空表达式求值 | 返回 0 |
| 括号不匹配 | 自动补全缺失的右括号 |

## 技术实现

- 纯前端单文件：HTML + CSS + JS，零依赖
- 表达式求值使用 `Function()` 构造器，支持 JS 原生运算符优先级
- CSS Grid 按钮布局，`aspect-ratio: 1` 保证按钮等宽
- 历史区滚动条在 Webkit 内核下显示 4px 细滚动条，Firefox 下使用 `scrollbar-width: thin`
